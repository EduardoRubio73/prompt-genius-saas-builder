Spec Técnica — Markdown

📋 Visão Geral

Projeto: ContractFlow — SaaS de Gestão de Contratos com IA
Stack: Next.js 15 + Supabase + Stripe + Resend
Público-alvo: PMEs brasileiras (5–50 funcionários)

🛠️ Stack Técnica

Frontend

  • Next.js 15 App Router + TypeScript strict
  • Tailwind CSS + shadcn/ui components
  • React Hook Form + Zod validation

Backend

  • Supabase (PostgreSQL + Auth + Storage + Edge Functions)
  • Stripe para pagamentos e assinaturas
  • Resend para e-mails transacionais

⚡ Funcionalidades Core

  • Assinatura Digital: Upload PDF + campos de assinatura + link seguro
  • Lembretes: Cron job via Supabase Edge Functions (30/7/1 dia antes)
  • Dashboard: Kanban de contratos por status + alertas de vencimento
  • WhatsApp: Integração via Evolution API ou Z-API

🔒 Segurança

  • RLS no Supabase — isolamento por organização
  • JWT com refresh automático
  • Arquivos no Supabase Storage com políticas privadas
